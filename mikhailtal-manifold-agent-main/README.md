# MikhailTal Manifold Agent

An open-source Python trading bot that participates exclusively in
Manifold Markets created by **MikhailTal**.

## Features
- Creator-restricted trading
- Probability calibration
- Fractional Kelly bet sizing
- Optional LLM-based judgment
- Clean, modular architecture

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python scripts/run_bot.py
